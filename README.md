# git 사용법

1. 로컬 저장소를 생성한다.
2. 코드를 작성한다.
3. 변경내역에 대해 commit을 생성한다. (commit은 버전이라고 생각) 
   1. 변경사항을 git이 감지한다. exept) .gitignire에 등로한 파일/디렉토리는 제외
   2. commit으로 남길 변경사항을 staging 역역에 추가한다.
   3. 메세지와 함께 commit을 작성한다.
